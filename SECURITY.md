# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in EpochLease, please report it responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

Instead, please email: 142044+tarurar@users.noreply.github.com

You should receive a response within 48 hours. We will work with you to understand
the scope and develop a fix before any public disclosure.
